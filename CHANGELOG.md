## Version Guide

### Versioning Scheme

Career Bridge AI follows **Semantic Versioning** (MAJOR.MINOR.PATCH):

- **MAJOR** (X.0.0): Breaking changes, major new features
- **MINOR** (1.X.0): New features, backward compatible
- **PATCH** (1.0.X): Bug fixes, minor improvements

### Release Cycle

- **Alpha (0.x.x)**: Development phase, foundation work
- **Beta (0.y.y)**: Feature complete, bug fixing phase
- **Release Candidate (0.z-rc)**: Ready for testing
- **Release (1.0.0+)**: Production ready
