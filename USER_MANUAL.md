## Getting Started

### What is Career Bridge AI?

Career Bridge AI is an intelligent platform that helps students:
- Optimize their resumes
- Discover suitable career paths
- Find scholarships and opportunities
- Learn new skills
- Plan their career journey

### System Requirements

- **Browser**: Chrome, Firefox, Safari, or Edge (latest versions)
- **Internet**: Stable connection required
- **Device**: Desktop, laptop, or tablet
- **Storage**: Minimum 100MB available

### Access the Platform

1. Open your web browser
2. Navigate to: `https://careerbridgeai.com`
3. Or run locally: `streamlit run app.py`

## Account Setup

### Creating an Account

#### Step 1: Sign Up
1. Click **"Sign Up"** on the homepage
2. Enter your email address
3. Create a strong password (minimum 8 characters)
4. Confirm password
5. Click **"Create Account"**

#### Step 2: Verify Email
1. Check your email for verification link
2. Click the link within 24 hours
3. Your email is now verified

#### Step 3: Complete Profile
1. Go to **Settings** > **Profile**
2. Add your details:
   - **Full Name**
   - **Current Education Level** (10th, 12th, Bachelor's, Master's, etc.)
   - **Field of Study**
   - **Career Interest**
   - **Location**
   - **Phone Number** (optional)

#### Step 4: Set Preferences
1. Go to **Settings** > **Preferences**
2. Choose:
   - **Notification Frequency** (Daily, Weekly, None)
   - **Preferred Job Locations** (Remote, On-site, Hybrid)
   - **Career Industries** of interest
   - **Scholarship Focus** (Merit-based, Need-based, Both)

### Logging In

1. Click **"Login"** on homepage
2. Enter email and password
3. Optional: Check **"Remember me"** for faster login
4. Click **"Sign In"**

### Password Management

#### Change Password
1. Go to **Settings** > **Security**
2. Click **"Change Password"**
3. Enter current password
4. Enter new password (minimum 8 characters, mix of letters/numbers/symbols)
5. Confirm new password
6. Click **"Update"**

#### Reset Forgotten Password
1. Click **"Forgot Password"** on login page
2. Enter your email
3. Check email for reset link
4. Create new password
5. You can now login with new password

## Resume Analysis

### Understanding Your Analysis

#### ATS Score
- **80-100**: Excellent (likely to pass ATS)
- **60-79**: Good (probably passes ATS)
- **40-59**: Fair (may be screened out)
- **Below 40**: Poor (likely to fail ATS)

**What it means**: ATS (Applicant Tracking System) score shows how compatible your resume is with automated screening systems used by companies.

#### Quality Metrics

**Overall Quality** (0-100)
- Comprehensive content assessment
- Includes formatting, structure, and content evaluation

**Formatting Score** (0-100)
- Resume layout and visual presentation
- Readability and organization
- Font and spacing consistency

**Content Score** (0-100)
- Relevance of information
- Achievement descriptions
- Keyword usage

### Skill Extraction

**What it shows:**
- Technical skills identified in your resume
- Soft skills detected
- Skill proficiency levels (when available)

**How to improve:**
- Be explicit about skills you mention
- Use industry-standard terminology
- List relevant technical certifications

### Improvement Suggestions

Career Bridge AI provides specific recommendations:

1. **High Priority** (🔴)
   - Critical issues affecting ATS
   - Example: Missing keywords, format issues

2. **Medium Priority** (🟡)
   - Important improvements
   - Example: Weak achievement descriptions

3. **Low Priority** (🟢)
   - Nice-to-have enhancements
   - Example: Additional certifications

### Implementation Tips

1. **Keywords**
   - Check job descriptions for target roles
   - Add relevant keywords naturally
   - Use industry-specific terminology

2. **Formatting**
   - Use standard fonts (Arial, Calibri, Times New Roman)
   - Maintain 0.5-1 inch margins
   - Use simple, clean layout

3. **Content**
   - Start bullet points with action verbs
   - Use quantifiable achievements
   - Keep to 1-2 pages

4. **Sections**
   - Include: Contact, Summary, Experience, Education, Skills
   - Optional: Projects, Certifications, Publications

## Learning Roadmaps

### Creating Your Roadmap

#### Method 1: From Career
1. Go to **Career Guidance** > **Recommendations**
2. Select target career
3. Click **"Create Learning Path"**
4. Review suggested roadmap

#### Method 2: Custom Roadmap
1. Go to **Learning** > **Roadmaps**
2. Click **"Create New Roadmap"**
3. Set **Target**: Career or skill
4. Set **Timeline**: 3-12 months
5. Choose **Pace**: Slow, Medium, Fast
6. System generates roadmap

### Understanding Your Roadmap

**Phases**
- Each roadmap divided into phases
- Typical phases: 4-8 weeks each
- Skills covered in each phase

**Learning Resources**
- **Courses**: Online/offline courses
- **Projects**: Hands-on projects
- **Books**: Reading materials
- **Communities**: Learning groups

**Progress Tracking**
- Mark courses as completed
- Track time spent
- Monitor skill improvement
- Celebrate milestones

### Course Types

**Free Courses**
- Open-source platforms
- University courses
- Community resources

**Paid Courses**
- Specialized training
- Certification courses
- Industry partnerships

**Estimated Cost**
- Free: ₹0
- Budget: ₹0-10,000
- Standard: ₹10,000-50,000
- Premium: ₹50,000+

### Project Ideas

Each roadmap includes project suggestions:

**Beginner Projects**
- Small, focused projects
- Build foundation skills
- 10-40 hours each

**Intermediate Projects**
- Larger scope
- Combine multiple skills
- 40-100 hours each

**Advanced Projects**
- Portfolio-worthy projects

### Adjusting Your Roadmap

1. Go to **Learning** > **Your Roadmap**
2. Click **"Customize"**
3. Modify:
   - **Timeline**: Change duration
   - **Pace**: Adjust learning speed
   - **Resources**: Add/remove courses
   - **Projects**: Select projects

## FAQs

### General Questions

**Q: Is Career Bridge AI free?**  
A: Core features are free. Premium features (if any) will have clear pricing.

**Q: Which browsers are supported?**  
A: Chrome, Firefox, Safari, Edge (latest versions). Mobile browsers supported.

**Q: Can I use it offline?**  
A: Some features require internet. Basic features work offline.

**Q: Is my data secure?**  
A: Yes. We use encryption and follow data protection standards. See [SECURITY.md](SECURITY.md).

### Resume Questions

**Q: What file formats are supported?**  
A: PDF, DOCX, DOC, and TXT files up to 10MB.

**Q: Why is my ATS score low?**  
A: Common reasons: Missing keywords, poor formatting, unclear content. Check improvement suggestions.

**Q: Can I upload multiple resumes?**  
A: Yes. You can upload and maintain multiple resume versions.

**Q: How often should I update my resume?**  
A: Every 3-6 months or after significant experience/skills gained.

### Career Questions

**Q: Are recommendations accurate?**  
A: Recommendations based on your skills and profile. Consider them suggestions, not definitive answers.

**Q: Can I change my target career later?**  
A: Yes. Update your profile anytime. Create new roadmaps for different careers.

**Q: How is salary data calculated?**  
A: Based on job market data, location, experience level, and industry trends.

### Scholarship Questions

**Q: Are all scholarships listed?**  
A: We cover major scholarships. Regional/niche scholarships may not be included.

**Q: When should I apply?**  
A: Apply as early as possible. Deadlines are strictly enforced.

**Q: What documents do I need?**  
A: Varies by scholarship. Check "Required Documents" section for each.

**Q: If rejected, can I apply again?**  
A: Usually only once per year. Some scholarships may allow reapplication next year.

### Learning Roadmap Questions

**Q: Are courses in the roadmap free?**  
A: Mix of free and paid. Roadmap shows cost for each course.

**Q: Can I skip phases?**  
A: Not recommended unless you already have those skills.

**Q: How long does the roadmap take?**  
A: Depends on your pace. Estimate provided at start.

## Troubleshooting

### Common Issues

#### Login Problems

**Issue**: "Invalid email or password"
- Solution 1: Ensure email is correct (case-sensitive)
- Solution 2: Use password reset if forgotten
- Solution 3: Check for extra spaces in credentials

**Issue**: "Email not verified"
- Solution 1: Check email for verification link
- Solution 2: Request new verification email
- Solution 3: Check spam folder

#### Resume Upload Issues

**Issue**: "File too large"
- Solution: Compress PDF or convert to lower quality
- Maximum: 10MB

**Issue**: "Unsupported format"
- Solution: Convert to PDF or DOCX
- Supported: PDF, DOCX, DOC, TXT

**Issue**: "Text not extracted"
- Solution: Ensure PDF is not image-based
- Try: Re-save PDF in text format

#### Display Issues

**Issue**: "Page won't load"
- Solution 1: Refresh page (Ctrl+R)
- Solution 2: Clear browser cache
- Solution 3: Try different browser

**Issue**: "Recommendations not updating"
- Solution 1: Re-analyze resume
- Solution 2: Update profile information
- Solution 3: Wait 24 hours for system update

2. **Search Help**: Use built-in search
